# Troubleshooting Guide

## General Taste Diagnosis

| If your coffee tastes... | The problem is likely... | Try adjusting... |
|--------------------------|--------------------------|------------------|
| Too sour/under-extracted | Under-extraction or underdeveloped | Grind 1-2 clicks finer, raise temp 1-2°C, pour slower |
| Too bitter/over-extracted | Over-extraction | Grind 1-2 clicks coarser, lower temp 1-2°C, pour faster |
| Weak/watery | Low extraction yield | Grind finer, use more coffee, or slower first pour |
| Too strong | Over-concentrated | Grind coarser, use less coffee, or add water post-brew |
| Muddy/cloudy | Too many fines or over-agitation | Use slower pours, let bed settle more, or filter |

## Processing-Specific Issues

**For Washed Coffees:**
- If tastes "papery" or "hollow" = possibly over-extracted, try coarser grind
- If tastes sharp/astringent = may need higher temp to develop sweetness
- If lacking clarity = grind finer, ensure good bloom

**For Natural Coffees:**
- If tastes "fermented" or "funky" beyond expected = over-fermentation, use less coffee or lower temp
- If tastes "jammy" or "wine-like" excessively = expected character, not a problem
- If bitter + sour simultaneously = likely uneven extraction, pour slower and more evenly

**For Honey Coffees:**
- If tastes "sticky" or "syrupy" excessively = try slightly higher temp
- If tastes unbalanced (sweet but hollow) = may need finer grind for more extraction

## Adjusting for Your Taste

- **Prefer brighter/acidic coffee:** Try lower temp (90-92°C), coarser grind, or shorter brew time
- **Prefer sweeter/milder coffee:** Try higher temp (93-96°C), medium grind, or longer brew time
- **Prefer stronger body:** Try coarser grind (for immersion methods) or slower final drawdown
- **Prefer cleaner cup:** Use paper filter, slower pours, or switch to V60/Chemex

## Common Mistakes by Brew Method

| Mistake | Result | Fix |
|---------|--------|-----|
| Pouring next pour before water drains (4:6) | Weak, under-extracted cup | Wait until water is nearly gone before next pour |
| Using too fine a grind (4:6) | Brew time exceeds 4:00, bitter | Go coarser; target 3:30 total |
| Using too coarse a grind (4:6) | Brew finishes before 3:00, weak | Go finer; target 3:30 total |
| Ignoring roast-level temperature (4:6) | Light roast tastes under-extracted at 87°C | Light roasts need 93-94°C; dark roasts need 83°C |
| Using standard AeroPress grind for inverted/long-immersion (2+ min, 20 g+) | Over-extraction: bitter, astringent, muddy | Go coarser by ~10 ticks (K-Ultra) or 2-4 steps (Encore ESP). Use HCG steep-and-release range, not standard AeroPress range. |
